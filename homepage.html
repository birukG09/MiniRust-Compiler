Features Tailored for Developers

Designed to be interactive, educational, and visually appealing, focusing on compiler concepts and Rust systems programming.

Simplified Rust Subset

Supports variables, functions, loops, conditionals, and basic types with Rust-like syntax.

Ownership & Borrow Checker

Catch ownership violations and borrow errors in real-time with descriptive hints.

AST & IR Visualization

Interactive tree views and flow graphs showing parsing, IR generation, and ownership flows.

REPL Playground

Type and execute MiniRust code live, with instant visual feedback.

LLVM Backend

Generates LLVM IR and optionally native assembly or WebAssembly for real execution.

Developer-Friendly UI

MoonSpace-GreenNature theme, smooth animations, syntax highlighting, and code diagnostics.

Why MiniRust Compiler?

It's more than just a compiler; it's a learning playground for developers:

  • Understand how Rust manages memory safely with ownership and borrowing.
  • Learn compiler design, parsing, IR generation, and LLVM integration hands-on.
  • Visualize code execution and catch errors early with intuitive diagnostics.
  • Experiment with optimizations, REPL, and even WebAssembly outputs.

Getting Started

Jump into exploring Rust internals in just a few simple steps

1

Clone the Repository

git clone https://github.com/your-username/mini-rust-compiler.git
2

Install Dependencies

cargo build

Install Rust packages and Inkwell bindings automatically.

3

Run REPL or Compile

cargo run --repl

Use REPL for live playground or --file example.rs to compile a file.
